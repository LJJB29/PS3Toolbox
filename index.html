Step 1: Install HFW (Hybrid Firmware)

Before you can install HEN, you need to install Hybrid Firmware (HFW). This is an official firmware + exploit enabler.

  1. Download the latest HFW (currently 4.92) from this link: Download HFW
  2. Extract the file and place the PS3/UPDATE/PS3UPDAT.PUP on a USB drive (formatted to FAT32).
  3. Insert the USB into your PS3 and go to: Settings > System Update > Update via Storage Media.
  4. Install the HFW like a normal update. Once complete, your system will still look official, but it's now ready for HEN.
Note: HFW 4.92 is required to install the latest PS3HEN. Make sure you're on 4.92 or update to it using the link above.

Step 2: Install PS3HEN

Once HFW is installed, visit this website on your PS3's Internet Browser to install PS3HEN:

URL: https://ps3toolbox.github.io

Then click below to launch the HEN installer:

Launch PS3HEN Installer
Pro Tip: Press Triangle on your PS3 browser > Tools > Homepage > Use Current to make this site your startup page.
